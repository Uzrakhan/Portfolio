## ✨ Overview

This portfolio was designed to go beyond the traditional developer website.

Instead of static sections and generic layouts, the experience focuses on:

- ⚡ High-performance interactions
- 🎨 Immersive visual storytelling
- 🌐 Real-time & motion-driven interfaces
- 🧠 Thoughtful frontend architecture
- 🖥️ Cinematic UI inspired by modern digital experiences

The project combines frontend engineering, animation systems, 3D rendering, and performance optimization into a single interactive portfolio experience.
